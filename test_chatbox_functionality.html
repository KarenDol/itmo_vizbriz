Dr. Briz Chatbox Enhancement Test

This page demonstrates the enhanced Dr. Briz chatbox functionality that has been implemented in the patient journey interface.

Enhanced Features:

  • Draggable: Click and drag the header to move the chatbox around the screen
  • Resizable: Drag the bottom-right corner to resize the chatbox
  • Always On Screen: The chatbox is constrained to stay within the viewport boundaries
  • Position Persistence: Position and size are saved to localStorage and restored on page reload
  • Minimize/Maximize: Use the control buttons to minimize or maximize the chatbox
  • Responsive: Automatically adjusts position when the browser window is resized
  • Visual Feedback: Hover effects and smooth transitions for better user experience
  • Z-Index Management: Brings to front when clicked or hovered
Note: The actual chatbox will appear in the bottom-right corner of the patient journey page. This test page is for documentation purposes only.
